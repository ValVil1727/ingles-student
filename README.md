# Simple Inventory System

## Description

This project is a simple Python program.  
It is designed to demonstrate basic programming concepts such as variables, user input, and calculations.

> The program is an inventory example that registers a product and calculates its total cost.  
It is a learning project created to practice explaining algorithms in English.

## How it works

> 1. The program asks the user for the product name.
> 2. It requests the product price from the user.
> 3. The program asks for the quantity of the product.
> 4. The system multiplies the price by the quantity.
> 5. The program prints the total cost of the product.

> The program checks the values entered by the user.  
> It processes the information and calculates the total value of the product.

## Status

> The project is currently running as a basic Python learning exercise.
